# NLP Learning Resources

A structured map of Natural Language Processing topics from foundations to modern LLMs.

## Topics

1. **[Probability & Information Theory](./Probability.md)** - Bayes' theorem, MLE vs MAP, Entropy, KL divergence
2. **[Markov Models & Sequence Modeling](./Markov.md)** - HMMs, Viterbi, N-gram models
3. **[Classical NLP Methods](./Syntax%20and%20Parsing.md)** - Tokenization, BoW, TF-IDF, POS tagging
4. **[Static Word Embeddings](./Static%20Embedding.md)** - Word2Vec, GloVe, FastText
5. **[Contextual Embeddings](./Contextual%20Embeddings.md)** - ELMo, subword tokenization
6. **[Word Embedding Properties](./Embedding%20Properties.md)** - Cosine similarity, bias evaluation
7. **[Lexical Semantics](./Lexical%20Sementics.md)** - WordNet, WSD, coreference
8. **[Classical ML for NLP](./Classical%20ML%20for%20NLP.md)** - Naive Bayes, SVM, CRF
9. **[Hidden Markov Models](./Hidden%20Markov.md)** - Forward/backward algorithms, Baum-Welch
10. **[N-gram Language Models](./N-gram.md)** - Smoothing, perplexity
11. **[Distributions](./Distributions.md)** - Gaussian, Categorical, Dirichlet
12. **[Perplexity](./Perplexity.md)** - LM evaluation metric
